Simplicity in the functionality

Userflow and wireframes are designed to test with the users to gauge how they interact with the web app, especially when the objective of the project was to design an analysis simulation tool. Our aim was to make the design as simple, friendly and easy to use so that it could help simplify the process of modeling the actual quantum experiments.

Linecons are designed

With simplicity in mind, we kept the web app simple and clean and not overly cluttered. Simple and easy-to-understand icons for Qucs's landing page were created to help explain the product.
